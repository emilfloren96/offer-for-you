# Project Structure

## Root files

- `package.json` - frontend dependencies and scripts
- `vite.config.ts` - vite build config
- `tsconfig.json` - typescript config
- `eslint.config.js` - eslint rules
- `.gitignore` - files ignored by git

## src/ - the frontend app

- `main.tsx` - entry point, mounts the react app
- `App.tsx` - root component, handles routing, global state and the landing page
- `index.css` - global css variables and base styles

### src/components/ - all ui components

- `Header.tsx` - top nav bar
- `ModelViewer.tsx` - the 3d house viewer where you click on parts
- `MaterialPanel.tsx` - shows materials and prices after clicking a part
- `CostSummary.tsx` - sticky bottom bar with the running total
- `OfferPanel.tsx` - the form you fill in to send an offer request
- `CompanyAuth.tsx` - login page for companies
- `CompanyDashboard.tsx` - dashboard for companies to see incoming offers
- `QuickPost.tsx` - modal for homeowners to describe a job in plain text

## src/data/ - static data

- `materials.ts` - all materials, prices, units and colors for each building part

## server/ - the backend

- `index.js` - express server with all api routes
- `db.js` - sqlite database setup and queries
- `products.db` - the actual database file with offers and company accounts
- `package.json` - server dependencies

## public/ - static files

- `models/` - all the 3d model files (.glb) for each house shape and floor count
- `vite.svg` - favicon

## dist/ - build output

generated when you run npm run build, not committed to git

## node_modules/ - installed packages

auto generated by npm, not committed to git
